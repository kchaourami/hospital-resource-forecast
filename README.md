# Hospital Resource Forecast – MVP
Develop a data-driven MVP to simulate and forecast hospital resource needs
for the Pitié-Salpêtrière hospital using synthetic data.

- Synthetic data generation based on public health statistics
- Time-series forecasting with Prophet
- Interactive dashboard for scenario simulation

- `data/` : datasets (raw & synthetic)
- `src/` : data generation & modeling code
- `notebooks/` : exploration & validation
- `dashboards/` : Streamlit MVP

- Python
- Pandas / NumPy
- Prophet
- Streamlit

All data used in this project is synthetic and does not contain any real patient information.
